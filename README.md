# Oxide Game of Life

This isnt a complex repo at all but more of me just testing Nvidias [Cuda-Oxide](https://github.com/NVlabs/cuda-oxide) implimentation. I think Rust as a language has a ton of value in many different areas, so seeing a solid cuda alternative to the traditional C/C++ apis made me very excited. This will just do some abritrary amount of itterations over an NxN sized array and spit out some time stats and an image of the final product. Over all pretty cool tech and I am super excited to see what can be made with it!
